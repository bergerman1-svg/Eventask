# Eventask

מערכת ניהול הפקת אירועים מודרנית בעברית, RTL, ללא תלויות חיצוניות.

## תכונות מרכזיות

- ניהול אירועים מרובים עם תאריכי התחלה וסיום
- לו"ז דינמי עם chain-shift אוטומטי בעריכה
- פעילויות נעולות (Go Live) ששוברות שרשרת
- מודל סטטוס משימה קולקטיבי (AND) — סטטוס מצרפי מכל המשתתפים
- ממשק שטח מותאם למובייל לאנשי צוות
- ניהול הפסקות צוות (מיידיות או מתוזמנות)
- העלאת קבצים למשימות
- ייצוא PDF של לו"ז עם לוגו
- שכפול אירוע ושמירה כתבנית
- אינטגרציה עם Google Calendar ו-Google Drive (UI מוכן)
- ציר זמן מולטי-יומי עם תצוגת WOW
- שעון מהיר (קפיצות של 5 דקות)
- תמיכה בהזמנת חברי צוות

## הפעלה מקומית

פתח את `index.html` בדפדפן מודרני (Chrome, Edge, Safari, Firefox). אין צורך בשרת — האפליקציה פועלת לחלוטין ב-frontend עם localStorage.

## פריסה ל-GitHub Pages

1. דחוף את הקוד לריפו ב-GitHub
2. הגדרות הריפו → Pages
3. בחר Source: Deploy from a branch
4. Branch: main, Folder: / (root)
5. Save
6. כתובת האתר תופיע תוך כמה דקות: `https://YOUR_USERNAME.github.io/REPO_NAME/`

## ארכיטקטורה

- Single-file HTML עם JavaScript וניל
- Persistence דרך localStorage
- RTL עברית עם תמיכה ב-LTR סלקטיבית (זמנים, תאריכים)
- ללא תלויות חיצוניות, ללא build step
- 5,644 שורות (HTML + CSS + JS משולבים)

## שימוש

1. פתח `index.html`
2. צור אירוע חדש (לוחצים על "+ צור אירוע חדש")
3. מלא תאריכים — ימים נוצרים אוטומטית
4. הוסף פעילויות לכל יום
5. הוסף משימות תחת פעילויות
6. הוסף חברי צוות
7. שתף את לינק ממשק השטח עם הצוות
